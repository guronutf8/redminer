Redmine time приложение для android для отправки затраеного времени на задачи.

Бонус: java библиотека для работы с сервером redmine info.guron.socket_redmine, позволяющая взаимодействовать с сервером redmine, полностью самостоятельная можно импользовать отдельно от основной программы. Из возможностей пока только загрузка списка задач, и отправка затраченого времени на сервер.
